# 비즈니스 매칭 시스템 사용자 설명 가이드

본 문서는 행사 운영자와 참가자에게 서비스를 설명할 때 사용할 수 있는 사용자용 안내문입니다. 개발 명세가 아니라 실제 이용 흐름을 기준으로 작성했습니다.

---

## 1. 서비스 한눈에 보기

이 서비스는 오프라인 비즈니스 매칭 행사를 운영하기 위한 웹 기반 매칭 관리 시스템입니다.

행사 운영자는 참가 스타트업과 전문가를 등록하고, 상담 시간표를 만들고, 예약 현황과 현장 진행 상황을 한 화면에서 확인할 수 있습니다. 스타트업은 등록된 연락처로 로그인해 전문가 상담을 예약하고, 전문가는 본인에게 배정된 상담 일정을 확인하며 상담 진행과 상담일지를 관리합니다.

핵심 목적은 다음과 같습니다.

1. 엑셀과 수기 연락으로 처리하던 매칭 운영을 줄입니다.
2. 스타트업 예약, AI 자동배치, 관리자 강제 조정을 한 흐름으로 관리합니다.
3. 현장 출석, 상담 진행 상태, 노쇼를 실시간으로 확인합니다.
4. 행사 종료 후 상담 기록과 만족도 데이터를 정리할 수 있게 합니다.

---

## 2. 사용자 유형

### 관리자

행사 전체를 설정하고 운영하는 사용자입니다. 행사 생성, 참가자 등록, 시간표 생성, 예약 현황 확인, AI 자동배치, 강제 배정, 현장 진행 모니터링을 담당합니다.

### 현장 스태프

행사 당일 현장에서 참가자의 출석을 확인하는 사용자입니다. QR 또는 검색 기반 출석 체크를 담당합니다.

### 전문가

상담을 진행하는 자문위원 또는 멘토입니다. 본인의 상담 시간표를 확인하고, 상담 시작/완료 상태와 상담일지를 관리합니다.

### 스타트업

행사에 참여하는 기업 사용자입니다. 등록된 이메일 또는 휴대전화로 인증번호를 받아 로그인하고, 상담 가능한 전문가와 시간대를 선택해 예약합니다.

---

## 3. 로그인 방법

### 스타트업과 전문가

1. 안내받은 로그인 URL에 접속합니다.
2. 역할 탭에서 `참가 스타트업` 또는 `전문가/자문위원`을 선택합니다.
3. 사전에 등록된 이메일 또는 휴대전화 번호를 입력합니다.
4. 전송된 6자리 인증번호를 입력합니다.
5. 인증이 완료되면 본인 역할에 맞는 화면으로 이동합니다.

인증번호는 일정 시간이 지나면 만료됩니다. 인증번호가 오지 않거나 만료된 경우 새 인증번호를 요청합니다.

### 관리자와 현장 스태프

1. 로그인 URL에 접속합니다.
2. 역할 탭에서 `운영진`을 선택합니다.
3. 운영 계정의 이메일과 비밀번호를 입력합니다.
4. 로그인 후 권한에 맞는 운영 화면으로 이동합니다.

---

## 4. 관리자 사용법

### 4.1 행사 만들기

1. 좌측 메뉴에서 `행사 목록`으로 이동합니다.
2. `행사 생성` 버튼을 누릅니다.
3. 행사명, 일정, 장소, 예약 가능 기간, 상담 시간 단위, 스타트업별 최대 상담 횟수를 입력합니다.
4. 저장하면 행사 목록에 새 행사가 표시됩니다.

행사는 운영 단계에 따라 `대기`, `예약`, `배치 조율`, `진행`, `종료`, `취소` 상태로 관리됩니다.

### 4.2 참가자 등록하기

1. 좌측 메뉴에서 `참가자 DB 관리`로 이동합니다.
2. `스타트업 목록` 또는 `전문가 목록` 탭을 선택합니다.
3. 참가자를 한 명씩 등록하려면 `개별 추가`를 사용합니다.
4. 여러 명을 한 번에 등록하려면 `CSV 일괄 업로드`를 사용합니다.
5. 등록 후 이메일, 휴대전화, 인증 가능 채널, 분야, 첨부 파일 정보를 확인합니다.

CSV 업로드 시 오류가 있는 행은 등록되지 않습니다. 오류 행 번호와 사유를 확인한 뒤 파일을 수정해 다시 업로드합니다.

### 4.3 행사에 참가자 배정하기

1. `행사 목록`에서 운영할 행사의 상세 화면으로 들어갑니다.
2. 행사 상태가 `대기`일 때 참가 스타트업과 전문가를 지정합니다.
3. 스타트업별 관심 분야와 전문가별 전문 분야를 확인합니다.
4. 행사장 테이블 정보를 등록하고, 필요하면 전문가별 기본 테이블을 지정합니다.

### 4.4 상담 슬롯 만들기

1. 행사 상세 화면에서 상담 시간대 생성 영역으로 이동합니다.
2. 행사 시작/종료 시간과 상담 단위를 기준으로 슬롯을 생성합니다.
3. 생성된 시간표에서 전문가별 빈 슬롯을 확인합니다.
4. 잘못 만든 미예약 슬롯은 정리 후 다시 생성할 수 있습니다.

### 4.5 예약 현황 확인하기

행사 상태가 `예약`이면 스타트업이 직접 상담을 예약할 수 있습니다.

관리자는 행사 상세 화면에서 다음 정보를 확인합니다.

1. 전체 예약률
2. 예약 완료 스타트업 수
3. 아직 예약하지 않은 스타트업 목록
4. 전문가별 빈 시간대
5. 예약 한도에 도달한 스타트업

### 4.6 AI 자동배치 사용하기

1. 행사 상태를 `배치 조율` 단계로 전환합니다.
2. 행사 상세 화면에서 `AI 자동배치` 화면으로 이동합니다.
3. 시스템이 미예약 스타트업과 빈 슬롯을 기준으로 배치 제안안을 만듭니다.
4. 분야 불일치, 시간 충돌 가능성, 미배치 사유를 확인합니다.
5. 필요한 경우 제안안을 조정합니다.
6. 문제가 없으면 제안안을 확정합니다.

확정 전 제안은 실제 시간표에 반영되지 않습니다. 확정 후에만 상담 슬롯에 반영됩니다.

### 4.7 강제 배정과 취소

현장 상황이나 긴급 변경이 생기면 관리자가 직접 슬롯을 조정할 수 있습니다.

1. 행사 상세 화면에서 조정할 슬롯을 선택합니다.
2. 스타트업과 전문가, 시간대를 확인합니다.
3. 강제 배정 또는 강제 취소 사유를 입력합니다.
4. 저장하면 변경 이력이 기록됩니다.

동일 시간대 중복 예약이나 테이블 충돌은 기본적으로 차단됩니다.

### 4.8 행사 당일 진행 모니터링

행사 상태가 `진행`이면 실시간 진행 화면을 사용합니다.

관리자는 다음 항목을 확인하고 처리합니다.

1. 시간대별 상담 진행 상태
2. 전문가별 테이블과 상담 기업
3. 스타트업 출석 여부
4. 전문가 출석 여부
5. 대기, 진행 중, 완료, 노쇼 상태

출석 상태가 잘못 입력된 경우 다시 미정 상태로 되돌릴 수 있습니다.

## 5. 스타트업 사용법

### 5.1 예약 화면 들어가기

1. 로그인 후 `내 예약 관리` 화면으로 이동합니다.
2. 참여 중인 행사가 여러 개라면 행사 선택 영역에서 원하는 행사를 선택합니다.
3. 현재 예약된 상담과 예약 가능 횟수를 확인합니다.

### 5.2 상담 예약하기

1. 예약 가능한 전문가 또는 시간대를 확인합니다.
2. 원하는 빈 슬롯을 선택합니다.
3. 상담 시간, 전문가명, 장소를 확인합니다.
4. 예약을 확정합니다.

행사별 최대 예약 횟수를 넘거나 같은 시간대에 이미 다른 상담이 있으면 예약할 수 없습니다.

### 5.3 예약 변경과 취소

1. `내 예약 관리`에서 기존 예약을 확인합니다.
2. 시간을 바꾸려면 `시간 변경`을 선택합니다.
3. 예약을 취소하려면 `예약 취소`를 선택합니다.
4. 변경 또는 취소 후 최신 예약 목록을 다시 확인합니다.

예약 기간이 끝난 뒤에는 직접 변경이 제한될 수 있습니다. 이 경우 운영진에게 문의합니다.

---

## 6. 전문가 사용법

### 6.1 오늘의 스케줄 확인하기

1. 로그인 후 `오늘의 스케줄` 화면으로 이동합니다.
2. 본인에게 배정된 상담 시간표를 확인합니다.
3. 현재 진행 중인 상담이 있으면 상단에 강조 표시됩니다.
4. 스타트업명, 기업 소개, 소개서, 테이블 위치를 확인합니다.

### 6.2 상담 진행 처리

1. 현장 도착 후 본인 출석을 체크합니다.
2. 스타트업 참석 여부를 확인합니다.
3. 상담이 시작되면 `상담 시작`을 누릅니다.
4. 상담이 끝나면 상담일지를 작성하고 완료 처리합니다.

### 6.3 상담일지 작성하기

상담일지에는 평가 점수와 상담 의견을 입력합니다.

평가 항목은 기술성, 전문성, 신뢰도, 협업 잠재력, 거래 가능성으로 구성됩니다. 점수는 관리자에게만 공개되며, 텍스트 의견은 공개 허용 여부에 따라 스타트업에게 공개될 수 있습니다.

---

## 7. 현장 스태프 사용법

1. 운영진 계정으로 로그인합니다.
2. `출석 체크` 화면으로 이동합니다.
3. QR 또는 참가자 검색으로 대상자를 찾습니다.
4. 스타트업 또는 전문가의 출석 상태를 체크합니다.
5. 잘못 체크한 경우 운영 관리자에게 정정 요청합니다.

## 8. 행사 상태별 이용 가능 기능

| 행사 상태 | 주요 의미 | 사용 가능한 기능 |
| :--- | :--- | :--- |
| 대기 | 행사 준비 단계 | 행사 정보 수정, 참가자 지정, 테이블 설정, 슬롯 생성 |
| 예약 | 스타트업 직접 예약 단계 | 스타트업 예약/변경/취소, 관리자 예약 현황 확인 |
| 배치 조율 | 예약 마감 후 조정 단계 | AI 자동배치, 관리자 수동 조정, 강제 배정 |
| 진행 | 행사 당일 운영 단계 | 출석 체크, 상담 진행 상태 확인, 노쇼 처리 |
| 종료 | 행사 완료 단계 | 만족도 확인, 상담 결과 정리, 데이터 추출 |
| 취소 | 취소된 행사 | 기존 이력 조회 중심, 주요 변경 제한 |

## 9. 자주 묻는 질문

### 인증번호가 오지 않습니다.

입력한 이메일 또는 휴대전화가 행사 등록 정보와 같은지 확인합니다. 계속 오지 않으면 운영진에게 연락해 등록 정보를 확인하거나 1회용 로그인 링크 발급을 요청합니다.

### 예약하려는 시간이 선택되지 않습니다.

이미 다른 상담이 같은 시간에 예약되어 있거나, 예약 가능 횟수를 초과했거나, 해당 슬롯이 마감되었을 수 있습니다.

### 예약 기간이 끝난 뒤 일정을 바꾸고 싶습니다.

예약 마감 후에는 직접 변경이 제한될 수 있습니다. 운영진이 행사 상세 화면에서 강제 조정할 수 있으므로 운영진에게 문의합니다.

### 전문가가 상담일지를 수정할 수 있나요?

행사 종료 전까지는 수정할 수 있도록 운영하는 것을 원칙으로 합니다. 최종 제출 후 수정 정책은 행사 운영 방침에 따릅니다.

### 현장에서 갑자기 참가자가 바뀌면 어떻게 하나요?

관리자가 참가자 정보를 수정하거나 세션을 무효화한 뒤, 필요하면 1회용 로그인 링크를 발급해 현장 예외 처리를 합니다.

## 10. 운영진 안내 문구 예시

### 참가자 공통 로그인 안내

안녕하세요. 비즈니스 매칭 행사 운영팀입니다.

아래 로그인 URL에 접속하신 뒤, 등록하신 이메일 또는 휴대전화 번호로 6자리 인증번호를 받아 로그인해 주세요.

- 로그인 URL: `{로그인 URL}`
- 스타트업: 로그인 후 `내 예약 관리`에서 상담 예약을 진행합니다.
- 전문가: 로그인 후 `오늘의 스케줄`에서 배정된 상담 일정을 확인합니다.

인증번호가 오지 않거나 로그인이 어려운 경우 운영팀에 문의해 주세요.

### 스타트업 예약 안내

예약 기간 동안 원하는 전문가와 상담 시간대를 직접 선택할 수 있습니다.

예약 화면에서 현재 예약된 상담과 남은 예약 가능 횟수를 확인한 뒤, 상담 가능한 슬롯을 선택해 예약을 완료해 주세요. 예약 마감 후에는 직접 변경이 제한될 수 있으므로 일정 변경이 필요하면 운영팀에 문의해 주세요.

### 전문가 현장 안내

행사 당일 로그인 후 `오늘의 스케줄`에서 본인의 상담 시간표를 확인해 주세요.

상담 시작 전 출석 상태를 확인하고, 상담 종료 후 상담일지를 작성해 주세요. 상담일지는 행사 결과 정리와 후속 매칭 검토에 활용됩니다.

## 11. 현재 구현 기준 메모

현재 구현 기준으로 관리자 핵심 화면인 행사 목록, 행사 상세, 참가자 DB 관리, AI 자동배치, 진행 현황 모니터링은 주요 기능이 구현되어 있습니다. 스타트업 예약 화면, 전문가 대시보드, 현장 스태프 출석 체크 화면은 운영 흐름상 설계가 정의되어 있으며 일부 화면은 단계적으로 구현될 예정입니다.
